Basic Car Components

Battery

Function: Supplies power to start the engine.

Type: Lead-acid, lithium-ion.

Engine

Function: Converts fuel into mechanical energy.

Type: Petrol, diesel, hybrid.

Steering System

Function: Controls the direction of the car.

Type: Power, manual.

Transmission

Function: Transfers power from engine to wheels.

Type: Manual, automatic.

Brakes

Function: Slows or stops the vehicle.

Type: Disc, drum.

Radiator

Function: Cools the engine.

Type: Liquid-cooled.

Alternator

Function: Charges the battery while running.

Type: Electric.

Axle

Function: Supports and rotates wheels.

Type: Front, rear.

Muffler

Function: Reduces engine noise.

Type: Exhaust system.

Car Suspension

Function: Absorbs shocks for comfort.

Type: Independent, dependent.

Catalytic Converter

Function: Reduces emissions.

Type: Three-way.

Power Train

Function: Transfers power to wheels.

Type: Hybrid, electric.

Braking System

Function: Ensures stopping safety.

Type: ABS, drum.

Camshaft

Function: Controls valve timing.

Type: Overhead, pushrod.

Exhaust System

Function: Directs engine gases.

Type: Single, dual.

Clutch

Function: Connects/disconnects engine and transmission.

Type: Hydraulic, cable.

Fuel Tank

Function: Stores fuel for engine.

Type: Metal, plastic.

Wheels

Function: Supports and moves the car.

Type: Alloy, steel.

Chassis

Function: Supports the entire car.

Type: Ladder, monocoque.

Cooling System

Function: Regulates engine temperature.

Type: Water-cooled, air-cooled.

Differential

Function: Distributes power to wheels.

Type: Open, limited-slip.

Pistons

Function: Moves inside the cylinder to generate power.

Type: Aluminum alloy.

Spark Plugs

Function: Ignites air-fuel mixture.

Type: Copper, platinum.

A/C Compressor

Function: Compresses refrigerant for cooling.

Type: Electric, belt-driven.
